Load file function



You can load a file from the system with this function.
Load File is useful when you want to read some (configuration) files (it's like LFI - Local File Inclusion), ex. "/etc/passwd", "/etc/shadow", etc.

NOTE:

If you have any trouble to run command on mysql shell, you must change the behavior of mysql to “TRADITIONAL MODE”.

a) To do it, type the following into the shell.

mysql> set session sql_mode='TRADITIONAL';



SELECT LOAD_FILE('/etc/passwd') into outfile '/var/lib/mysql-files/passwd.txt';


Output:


b) Check permission to read and write in “/var/www/sqli-labs.com/public_html”.

$ cd /var/www/sqli-labs.com/public_html
$ le -a



Output:


If you don't have all permissions run the following (Only for testing not production environments).

$ chmod -R 777 /var/www/sqli-labs.com/public_html
