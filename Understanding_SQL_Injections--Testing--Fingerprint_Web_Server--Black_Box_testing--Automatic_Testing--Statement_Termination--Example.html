Example



We have this query.

select top 1 name from color
where colorid=4
order by name


This query is not terminated.

If I add a ‘;' at the end then, It's terminated and I can add another query.

select top 1 name from color
where colorid=4
order by name; select * from city


The semicolon allows to terminat one query and execute another.

But if we type the ‘;’ before finishing the query, then there will be an error.

select top 1 name from color;
where colorid=4
order by name


If we run the above code will get an exception.
