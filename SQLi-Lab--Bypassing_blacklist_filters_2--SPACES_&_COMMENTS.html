MySql is a very flexible database and it allows us to use different arbitraty inputs in place of the nomal ones. For instance:

• Instead of “SPACES” we can use (‘%09’, ‘%0A’, ‘%0C’, ‘%0D’, ‘%0B’ ‘%a0’).
• Instead of “+” we can use some non-printable characters which get represented as spaces at the backend.
