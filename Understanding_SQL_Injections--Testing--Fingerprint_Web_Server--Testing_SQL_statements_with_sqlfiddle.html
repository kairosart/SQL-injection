Testing SQL statements with sqlfiddle



In order to test our SQL queries we're going to use “sqlfiddle”. To access go to http://sqlfiddle.com/ in your web browser.



On the left panel you can create your schema. For example:

CREATE TABLE `users` (
  `id` INT NOT NULL AUTO_INCREMENT,
  `email` VARCHAR(45) NULL,
  `password` VARCHAR(45) NULL,
  PRIMARY KEY (`id`));
  
  
insert into users (email,password) values ('m@m.com',md5('abc'));


Insert your code and click on “Build Schema” button.

On the right panel you can build your SQL queries. For example:

select * from users; 


Insert your query and click on “Run SQL” button.

You see at the bottom the response of your query.
