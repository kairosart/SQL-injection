Finding columns used by developers



We're going to try and find the number of columns used in the query.

To do this we can't use a SQL Keyword “ORDER BY”.

NOTE: "ORDER BY" doesn't work if you're not using comments.

In this case we have to do a kind of try and error or Force brute.

a) Let's try with 3 columns.

Insert: ORDER BY 3.




Result: Unknown column '3' in 'order clause'

b) Let's change the number or columns.

Insert: ORDER BY 2.



Result: We don't get an error, so there are 2 columns.

NOTE
We have successfully derived the query. So the knowledge we take out from this result is that the developer has used the query
