Using the SQL comment syntax



For the previous section we have the following query.

select top 1 name from color;
where colorid=4
order by name


This gives us an expection.

To avoid this exception we can use the ‘comment syntax’.

• One way would be adding ‘--’ at the begining of the rest lines.

select top 1 name from color;
--where colorid=4
--order by name


• The other one is adding ‘/*’ at the beginning of the text to comment and ‘*/’ at the end.

select top 1 name from color;
/*where colorid=4
order by name*/
